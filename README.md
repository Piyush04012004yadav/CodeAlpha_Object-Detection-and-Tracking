# CodeAlpha_Object-Detection-and-Tracking
# 🎯 Real-Time Object Detection & Tracking System

A high-performance computer vision project that performs **real-time object detection and multi-object tracking** using **YOLOv8** and **Deep SORT**.

---

## 🚀 Features

* 🔍 Real-time object detection using YOLOv8
* 🎯 Multi-object tracking with unique IDs
* 📦 Bounding box visualization
* ⚡ Fast and lightweight implementation
* 🎥 Works with webcam or video input
* 🧠 Scalable for advanced AI applications

---

## 🧰 Tech Stack

* **Language:** Python
* **Libraries:** OpenCV, NumPy
* **Detection Model:** YOLOv8 (Ultralytics)
* **Tracking Algorithm:** Deep SORT

## 🎮 Usage

* The webcam will open automatically
* Objects will be detected in real-time
* Each object will be assigned a **unique tracking ID**
* Press **ESC** to exit

---

## 🧠 How It Works

1. Video frames are captured using OpenCV
2. YOLOv8 detects objects in each frame
3. Detection results are passed to Deep SORT
4. Deep SORT assigns unique IDs and tracks objects
5. Output is displayed with bounding boxes and IDs
